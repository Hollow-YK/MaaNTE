# 喷泉打卡

## 简介

自动识别并完成喷泉打卡的每日签到功能。

需要使用前台控制器。

## 功能

自动依次完成以下操作：

1. **识别喷泉打卡按钮** — 通过模板匹配识别打卡交互按钮，按 F 交互
2. **等待动画** — 等待打卡动画播放结束
3. **许愿** — 识别许愿选项并点击，完成许愿
4. **跳过剧情** — 自动跳过许愿过程中的剧情对话
5. **退出** — 按 Esc 退出打卡界面

该任务无可配置选项，启动后即完整运行全部流程。

## 配置详解

该任务无子选项，无需额外配置。

**具体实现**：任务入口 `FountainCheckinEntrance`，无 `option` 字段。使用模板匹配（`TemplateMatch`）识别喷泉打卡按钮和许愿选项，控制器限制为 `Win32-Front`。
